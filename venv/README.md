\# 🤖 AI Resume Analyzer Pro



An intelligent resume analysis tool that uses \*\*Natural Language Processing (NLP)\*\* and \*\*Machine Learning\*\* to match resumes against job descriptions.



\## 🎯 Features



\- 📄 PDF Resume Upload \& Text Extraction

\- 🧠 TF-IDF Vectorization + Cosine Similarity Matching

\- 🎯 Automatic Skill Extraction (100+ skills)

\- 📊 Interactive Visualizations (Gauge, Pie, Bar Charts, Word Cloud)

\- 📥 Downloadable PDF Reports

\- 🏆 Multi-Resume Comparison \& Ranking

\- 🤖 AI-Powered Improvement Suggestions



\## 🛠 Tech Stack



\- \*\*Language:\*\* Python 3.x

\- \*\*Framework:\*\* Streamlit

\- \*\*ML/NLP:\*\* scikit-learn, spaCy

\- \*\*Visualization:\*\* Plotly, Matplotlib, WordCloud

\- \*\*PDF Processing:\*\* PyPDF2, FPDF
